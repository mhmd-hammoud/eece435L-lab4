## Tkinter Interface

To run the Tkinter interface, run the following command in the terminal:

```bash
python3 tkinter_interface.py
```

The Tkinter interface will open up. You can enter a number in the entry box and click the "Calculate" button to see the factorial of the number.

## PyQt5 Interface

To run the PyQt interface, run the following command in the terminal:

```bash
python3 pyqt_interface.py
```

The PyQt interface will open up. You can enter a number in the entry box and click the "Calculate" button to see the factorial of the number.
